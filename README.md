# Python-calculator
Explainattion:
a simple calculator built using python in jupyter notebook.
It allows the user to enter two numbers and select an arithmetic operation.
The program uses input() to take values from the user.
The inputs are converted to float so decimal numbers can be used.
Conditional statements (if, elif, else) are used to decide which operation to perform.
The program handles division by zero and invalid operators.
Clear messages are shown for both results and error.
